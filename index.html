Bio

I am a developer with a passion for security and open source.

I started to use linux when I was in middle school and have been learning ever since. I also became fasinated with the web and how it is a platform for delivering ideas and experinces. In highschool I took some programming classes and worked as a web developer, during this job I learned python and the django framework, which let me make data driven web sites. I also used the scrapy web scraping framework to automatically collect the data for the sites.

When I entered college at UNCC my passion for security was found when I joined the 49th security division. I participated in many ctf's including csaw, ictf, ructf, luctf, polictf, and others. I halve also helped orginize ctfs with the club for the university's cyber security symposium, and for infraguard.

I also do security research with other security professionals at whiskey sec.
